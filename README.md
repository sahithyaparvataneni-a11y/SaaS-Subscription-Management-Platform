This is a SaaS Subscription Management Platform built using HTML, CSS, Bootstrap, JavaScript, Node.js, Express, and MySQL. It allows users to register, login, and access a protected dashboard where they can view their profile details and choose subscription plans like Basic, Pro, and Enterprise. The project demonstrates core SaaS features such as authentication using JWT, session handling with localStorage, and a simple plan management system, making it a beginner-friendly full-stack web development project.
